About

Parrot Zik is a great set of headphones with a good application for mobile devices.

Zik Controller is a third party application for the Mac platform so you can control your Zik with your Mac.

It has feature parity with the mobile applications from Parrot except for the firmware upgrade which we left for Parrot to handle since it is a critical operation.

You can check the battery status as well as control the noise cancellation feature and other DSP effects.
